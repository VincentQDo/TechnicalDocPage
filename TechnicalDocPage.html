Introduction

Welcome to introduction to programming.

In this documentation you will learn basic coding concepts in Python

variables

variables are containers that can be used to store data. Let's look at this piece of code:

name = "Huyen";
age = 20;

This will let you easily replace the value in a program without retyping it too many times. For example you can do something like this:

name = "Huyen"
age = 20
print("My name is " + name + " and I am " + age + " years old.");

The above code snippet will set the variable name to "Huyen" and age to 24, then it will print out the sentence "My name is Huyen and I am 20 years old". The computer automatically know to replace the variable names with the value of that variable.

if...else... statements

if else conditional statements will allow you to executes block of codes when a certain condition is met. for example:
if x > 5:
print("x is greater than 5")
else:
print("x is less than 5")

The above code snippet will print out "x is greater than 5" if the variable x is bigger than 5, other wise the program will print out "x is less than 5".

while loop

while loop are used to repeat a section of the code. When a programmer wants to do something multiple times, they can write a while loop to repeat the same action without having to repeating themselves.
while x < 10:
  print("Hello")

Using the while loop will save you a lot of typing repetive code, but beware that sometimes while loop will go forever if there is an error in the condition, that will make the program crash.
while true:
  print("Hello")
The code above will keep going indefinitely because the computer does not know when to stop since we never tell it to.

for loop

For loop is similar to while loop, it allows programmer to repeat a section of code a certain number of times. unlike while loop howerver, the for loop is safer because it will only repeat a certain amount of times and then it will terminate.
for i in range(6):
  print("Hello")

The code above will print Hello 6 times. in Python, range(6) will generate a list of numbers from 0 to 5 and then the for loop will iterate over that list and execute the statements in the for loop until it run out of items to iterate.
